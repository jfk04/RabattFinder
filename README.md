# 🛒 RabattFinder

## 📌 Introduction
This mobile application helps you to find discounts for the product you desire. Simply enter the name of a product and the app will tell you in which stores you wil get a discount for the given product.

## 🛠 Technnologies I used
Only javascript/node.js <br>
For the UI I used React Native with Expo and for the backend I used express <br> to set up a server. To get the data from the shops I used axion and cheerio <br> to scrape the sites.

## ✨ Features
- 🔍 Search and track specific food items
- 🏪 Choose the supermarkets you want to scan
- 💸 Detect discounts and price drops
- 🔔 Get notified when products go on sale
